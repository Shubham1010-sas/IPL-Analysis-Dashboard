# 🏏 IPL Analysis Dashboard 2008–2019

## Project Overview
End-to-end data analysis project on 756 IPL matches using Excel, SQL, Python and Power BI.

## Tools Used
- Excel (Data Cleaning & Pivot Tables)
- SQL / DB Browser SQLite (Business Queries)
- Python - Pandas, Matplotlib, Seaborn (Data Analysis & Charts)
- Power BI (Interactive Dashboard)

## Key Insights
- Mumbai Indians are the most successful team with 109 wins
- Teams prefer fielding after winning toss (61% vs 39%)
- CH Gayle won the most Player of the Match awards
- IPL-2013 had the highest number of matches (76)
- Winning the toss does NOT guarantee winning the match
